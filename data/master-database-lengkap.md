# Master Database Lengkap — Jelajah Jogja

Single source of truth untuk seed data lengkap.

**Berisi 3 tabel utama:**
1. **Tabel 1 — Atribut Operasional Destinasi** (17 kolom): untuk seed tabel `Destination` di Prisma
2. **Tabel 1B — Atribut Detail untuk Halaman Detail** (6 kolom): untuk seed kolom tambahan di `Destination` (alamatLengkap, jamBuka, jamTutup, tipsRombongan, deskripsiPanjang)
3. **Tabel 2 — Matriks Skor TOPSIS** (41 kolom): untuk seed tabel `DestinationScore` (75 × 40 = 3.000 baris)

**Untuk data Hotel & Kuliner**: lihat `master-data-hotel.md` dan `master-data-kuliner.md` di folder ini.

## Cara Pakai untuk Seeding

Di `prisma/seed.ts`, parse file ini lalu seed dua tabel:

```typescript
import fs from 'fs';
// Parse Tabel 1 → seed ke tabel Destination
// Parse Tabel 2 → loop 40 subkriteria per destinasi → seed ke DestinationScore

for (const dest of destinations) {
  const d = await prisma.destination.create({ data: { ...dest } });
  // Untuk tiap 40 subkriteria di Tabel 2:
  for (const [subCriteriaKey, scoreValue] of Object.entries(scoresFor(dest.id))) {
    const [kriteria] = subCriteriaKey.split('_'); // K1_S1 → K1
    await prisma.destinationScore.create({
      data: {
        destinationId: d.id,
        kriteriaKey: kriteria,
        subCriteriaKey,
        scoreValue: parseInt(scoreValue),
      },
    });
  }
}
```

Total hasil seed: 75 baris di `Destination` + 3.000 baris di `DestinationScore`.
